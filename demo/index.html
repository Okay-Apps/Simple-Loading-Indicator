Simple Loading Indicator

What is it?

It's a JavaScript file that has show and hide functions. Show function shows a full page overlay with the text (Loading...) in the middle and hide function hides it.

In this demo we are calling hide function 5 seconds after show.

How to use it?

Include the .js file or .min.js file in your HTML file then call showLoading() to show the loading overlay with text, to hide it, call hideLoading() and thats it.

Go to GitHub repo: https://github.com/Okay-Apps/simple-loading-indicator.js
